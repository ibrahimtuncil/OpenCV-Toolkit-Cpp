# OpenCV-Toolkit-Cpp (v1.0)

A high-performance foundational toolbox built with C++ and OpenCV for image and video manipulation.

## Features
- **Input Validation:** Automatically sanitizes file paths (removes quotes, converts `\` to `/`).
- **Media Support:** Handles Images, Videos, and Webcam streams.
- **Interactive Video:** Change FPS on the fly using keyboard inputs.
- **Drawing Tools:** Functions for shapes, text overlay, and ROI cropping.

## Tech Stack
- **Language:** C++
- **Library:** OpenCV 4.x
- **Focus:** Performance and Memory Efficiency

## Roadmap
- Matrix Operations & Pixel Manipulation
- Mouse Callbacks (ROI Selection)
- Trackbars for Dynamic Filtering
- Color Space Conversions (RGB, HSV, Gray)
